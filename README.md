# RIKKO

[![License: MIT](https://shields.io/badge/LICENSE-MIT-green)](LICENSE)
[![Cross Platform](https://shields.io/badge/Cross%20platform-blue)](https://github.com/yash12007-com/rikko.git)

A whole new programming language for developers to create their dream application software with minimal code. The language RIKKO was developed by Yash12007, a software company, in 2026. The language template has been ported from Python to Node.js (Electron framework) for more efficient compile-time. The porting of the language is still in progress.

## Pre-release Announcements
The language source code is changing to open source for developers to understand the logic behind the execution, and the source of the interpreter will be closed for security reasons. The completion of the language source code will result in the release of NovaStream IDE. Specifically designed for the RIKKO programming language.
- Pros
  - The language is specifically designed for (.rik) RIKKO files, where developers can write their code of a minimum of 4 lines to create their first application.
  - RIKKO is made for easy code execution and the fastest compilation for developers and end-users.
  - The language source code is being ported to Electron for dynamic software applications and user interfaces, including the user experience.
  - The interpreter of the RIKKO language works without setting up any environment variable on the device by directly executing a script with the interpreter using the CLI.
  - RIKKO is a cross-platform language.
  - The language is open-source to everyone, and anyone can collaborate

- Cons
  - The language is still being ported, which means some minimal warnings can appear at runtime.
  - The interpreter is closed source, which means that the developer should use the compatible device according to the release or the device with the minimum SDK needed.
  - Developers should use the given syntax to avoid runtime or compile-time errors due to a case-sensitive codebase
